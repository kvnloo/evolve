# Research Synthesis

Polished research findings ready for human consumption.

---

## Purpose

This folder contains synthesized research that has been:
1. Validated and verified
2. Organized for readability
3. Distilled into actionable insights

---

## Lifecycle

```
claudedocs/active_research/   →   docs/research-synthesis/   →   claudedocs/archive/
        (raw research)              (polished findings)            (historical)
```

---

## When to Add Content Here

Add research to this folder when:
- Research is complete and conclusions are validated
- Findings have been synthesized into actionable insights
- Documentation is ready for team consumption
- Content would benefit from human-readable formatting

---

## Content Guidelines

1. **Clear summaries**: Lead with key findings
2. **Actionable insights**: Focus on what can be done with the research
3. **Cross-references**: Link to raw research in active_research/ if needed
4. **Visual aids**: Use diagrams and tables where helpful
